
# Windows Commands

### Find open ports of machine
  
  - Run command in cmd
      `
          netstat -aon
	`

  
  - Sample output

>      
>       Proto  Local Address          Foreign Address        State           PID
>       TCP    0.0.0.0:8080           0.0.0.0:0              LISTENING       9976
>       TCP    0.0.0.0:8009           0.0.0.0:0              LISTENING       9976
